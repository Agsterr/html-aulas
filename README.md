# HTML Aulas

Exercícios práticos de **HTML e CSS** do curso da Alura — páginas estáticas para aprender estrutura, semântica e layout.

---

## O que contém

- `index.html` — página principal com header, navegação, seções e rodapé
- `css/style.css` — estilização com fontes Google (Montserrat, Open Sans)
- `projeto 4/` — variação do projeto com layout próprio
- `imagens/` — assets utilizados nas páginas

---

## Tecnologias

- HTML5
- CSS3
- Google Fonts

---

## Como visualizar

```bash
git clone https://github.com/Agsterr/html-aulas.git
cd html-aulas
```

Abra `index.html` no navegador ou use a extensão **Live Server** no VS Code.
